Pour aller toujours plus loin : s’entraîner et coacher sur les postures relationnelles, voici la dernière clé de votre parcours : Mobile Practice, votre application de coaching vidéo.

Elle est :

  • Simple à prendre en main.
  • Idéale pour accompagner ses équipes ou s’entraîner
  • Parfaite pour être réactif ou recevoir du feedback personnalisé
  • Pleine de ressources (écrits, vidéos) pour que chacun puisse exprimer sa considération pour le travail effectué lors des entraînements de ses collaborateurs ou exprimer sa personnalité dans les appels clients.

Mobile Practice est une application que vous pouvez utiliser sur Mobile pour une meilleure expérience.


Si vous êtes sur iPhone, téléchargez Mobile Practice sur l’App Store, et suivez le guide.

Si vous êtes sur Android, activez votre compte

et connectez-vous sur le site https://app.mobilepractice.io/
